# 🎮 Sokobana

**Vanilla JavaScript · 2024 · Collège de Maisonneuve**

A browser-based puzzle game inspired by the classic Sokoban (1982), where the player pushes boxes onto target tiles to complete each level.

🔗 Live Demo: https://paticodes.github.io/Sokabana-game/

---

## 🧩 Overview

Sokobana is a collaborative school project built with @TekGeek_dev for the *Interface Programming 2* course.

The objective was to go beyond the assignment requirements and create a game that feels polished, engaging, and visually cohesive while also reimagining the character as a female protagonist to bring more representation into tech and gaming.

---

## 🎯 The Challenge

The project required building a complete game using **Vanilla JavaScript only**, with a strong focus on:

- Object-Oriented Programming (OOP)
- Class structure (methods & properties)
- DOM manipulation (no frameworks)

Core challenges included:

- Player movement system
- Collision detection
- Box pushing logic (Sokoban rules)
- Managing state across 10 levels

### Technical Structure

The game is built around:

- **Level class**
  - Parses level data into a 2D grid
  - Handles movement logic and rules

- **Main module (main.js)**
  - Manages keyboard input
  - Handles rendering
  - Controls level progression

Each game element is represented using a single-character system:
- Player
- Box
- Wall
- Target
- Box on target

The entire board re-renders dynamically on each key event.
